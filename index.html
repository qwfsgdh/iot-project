Smart Shoes for Blind People

An intelligent IoT-based assistive navigation system that empowers independence and confidence for visually impaired individuals.

Explore Project ↓ Impact

Why This Project Matters

Understanding the real-world challenges and the innovation behind our assistive navigation system

The Problem

Navigation tools for visually impaired people are often bulky, uncomfortable, and draw unwanted attention. Traditional white canes cannot detect elevated obstacles, and many assistive devices feel stigmatizing.

Our Solution

A discreet, ergonomic system integrated directly into footwear. Smart shoes scan the environment, detect obstacles, and notify users through intuitive haptic feedback — all without drawing attention.

How the System Works

Obstacle Detection

Ultrasonic and LIDAR sensors continuously scan 15–150 cm in front of the user, detecting ground and upper-level obstacles.

Adaptive Signal System

ESP32 processes sensor data in real-time, adjusting vibration feedback intensity as obstacles get closer.

GPS Emergency Tracking

GPS NEO-6M enables emergency location broadcasting with ±5m accuracy and optional WiFi/Bluetooth transmission.

Technical Workflow

Environmental Sensing

HC-SR04 and TFmini Plus LiDAR scan a 120° field-of-view with millisecond updates.

Data Processing

ESP32 analyzes incoming sensor streams and filters noise before classification.

Location Tracking

GPS NEO-6M maintains continuous positioning and supports emergency alerts.

Feedback Generation

Vibration motors + buzzer output layered notifications depending on urgency.

Detailed Technical Specifications

Complete hardware and software specifications of the smart-shoe navigation system

Processing Unit

  • ESP32-WROVER-B Module
  • Dual-core Xtensa LX6 @ 240MHz
  • 520KB SRAM, 4MB Flash
  • Wi-Fi 802.11 b/g/n (2.4GHz)
  • Bluetooth 5.0 + BLE
  • Real-Time OS (FreeRTOS)

Sensor Array

  • HC-SR04 Ultrasonic Sensor (40kHz)
  • TFmini Plus LiDAR
  • GPS NEO-6M (±5m accuracy)
  • IMU & 3-Axis Accelerometer
  • Detection Range: 15–150cm
  • 120° Horizontal Field of View

Feedback Systems

  • Piezo Buzzer (80Hz–16kHz)
  • 3× Directional Vibration Motors
  • Optional Speaker Output
  • Adaptive Intensity Control
  • Multi-Pattern Alert Modes
  • Response Latency: <100ms

Power Management

  • 3000mAh Li-Po Battery
  • 8+ Hours Operation
  • Fast Charging (≈2 hours)
  • Dynamic Power Optimization
  • Low Battery Warning System
  • Efficiency: >99%

Physical Specifications

  • Weight: <200g per shoe
  • Water Resistance: IP54
  • Operating Temp: −10°C to +50°C
  • Ergonomic In-Shoe Form Factor
  • Durable ABS Housing
  • Modular Replaceable Parts

Software Features

  • Real-Time Obstacle Classification
  • Edge-AI Processing
  • Adaptive Filtering Algorithms
  • GPS Data Logging
  • Emergency Alert Broadcasting
  • OTA Firmware Updates
